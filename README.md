Playwright Test Framework (JavaScript)
Framework de automatización de pruebas diseñado con Playwright y JavaScript para garantizar la calidad en entornos web.

🚀 Tecnologías
Framework: Playwright

Lenguaje: JavaScript

Estrategia: Page Object Model (POM)

🛠️ Instalación
Para ejecutar este proyecto en tu máquina local:

Clonar el repositorio:

Bash
git clone https://github.com/geniol57/playwright-saucedemo-framework.git
Instalar dependencias:

Bash
npm install
Instalar los browsers de Playwright:

Bash
npx playwright install
🧪 Ejecución
Para ejecutar las pruebas, utilizá el siguiente comando:

Bash
npx playwright test

📋 Sobre el Proyecto
Este proyecto demuestra la implementación de flujos de automatización robustos y escalables utilizando JavaScript, enfocados en la mantenibilidad mediante el patrón Page Object Model.
